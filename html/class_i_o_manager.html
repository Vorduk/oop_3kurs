Реализация менеджера ввода/вывода

Управляет всеми датчиками и исполнительными устройствами в системе. Хранит объекты в ассоциативных контейнерах для быстрого доступа по ID.

Датчики хранятся в map<int, shared_ptr<ISensor>> для доступа по ID Устройства хранятся в двух контейнерах: по ID и по типу Поддерживает команды трёх типов: On/Off, мощность, режим

Заметки
Использует умные указатели для автоматического управления памятью

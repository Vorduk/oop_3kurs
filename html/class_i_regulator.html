Подробное описание

Интерфейс для алгоритмов регулирования

Определяет интерфейс для всех алгоритмов, вычисляющих управляющий сигнал на основе отклонения текущего значения от целевого.

Методы

pure virtual

Рассчитать управляющий сигнал

Вычисляет значение управляющего воздействия на основе текущего и целевого значения параметра.

Аргументы
currentТекущее значение параметра (например, 22.5 °C)
targetЦелевое значение параметра (например, 23.0 °C)
Возвращает
Управляющий сигнал в процентах (0-100), где: 0 - устройство выключено, 100 - максимальная мощность/интенсивность

Замещается в OnOffRegulator и PidRegulator.

Сбросить внутреннее состояние регулятора

Необходим для регуляторов, накапливающих историю (например, PID). Вызывается при смене целевых параметров или перезапуске системы.

Переопределяется в PidRegulator.

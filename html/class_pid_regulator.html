Подробное описание

ПИД-регулятор (Proportional-Integral-Derivative) для плавного управления параметрами

Принцип работы: ПИД-регулятор вычисляет управляющий сигнал (0-100%) на основе разницы между текущим и целевым значением (ошибки). Сигнал состоит из трёх компонентов:

Пропорциональный (P): реагирует на текущую ошибку. Чем больше ошибка, тем сильнее воздействие. Интегральный (I): накапливает ошибку во времени, устраняет установившуюся ошибку. Дифференциальный (D): реагирует на скорость изменения ошибки, сглаживает колебания.

Формула: output = P * error + I * ∫error dt + D * d(error)/dt

Аргументы
kpПропорциональный коэффициент (P). Лучше диапазон: 0.5-5.0 Определяет, насколько сильно регулятор реагирует на текущую ошибку. Слишком высокое значение вызывает колебания, слишком низкое - медленную реакцию.
kiИнтегральный коэффициент (I). Лучше диапазон: 0.01-0.5 Определяет скорость устранения установившейся ошибки.
kdДифференциальный коэффициент (D). Лучше диапазон: 0.1-2.0 Определяет демпфирующий эффект, предсказывая будущую ошибку. Помогает предотвратить колебания.
dtВременной шаг в секундах между вызовами calculate() Должен соответствовать частоте вызова регулятора

Вычисление управляющего сигнала на основе текущего и целевого значения

Алгоритм:

  1. Вычисление ошибки = целевое - текущее
  2. Пропорциональная составляющая: P = Kp * error
  3. Интегральная составляющая: I += error * dt, затем I = Ki * I
  4. Дифференциальная составляющая: D = Kd * (error - prevError) / dt
  5. Суммирование: output = P + I + D
  6. Ограничение выхода в диапазоне [0, 100]

currentТекущее значение параметра (температура, влажность и т.д.)

Управляющий сигнал в процентах (0-100), где 0 = выключено, 100 = полная мощность

Подробное описание

Интерфейс поставщика данных о состоянии теплицы

Предоставляет доступ к текущим параметрам микроклимата: температура воздуха, влажность воздуха и влажность почвы. Этот интерфейс используется датчиками для получения актуальных показаний. Реализуется классом SimulationModel, который симулирует физические процессы.

Заметки
Все методы константные, так как чтение данных не должно изменять состояние
